Makakoo OS ships more than 200 built-in AI skills organized across 13 categories. Each skill is a plugin of kind `skill` — a `plugin.toml` manifest paired with a `SKILL.md` that describes when and how to invoke it. Skills are installed individually with `--core` or in bulk via a distro.

## Installing and running skills

```bash
# Install a skill from the core library
makakoo plugin install skill-research-arxiv --core

# Run a skill directly
makakoo skill research-arxiv

# Check what's installed
makakoo plugin list

# Get full manifest and lock entry for a skill
makakoo plugin info skill-research-arxiv
```

<Tip>
  Skills of kind `skill` are passive — they are not managed by the daemon. Running `makakoo skill <name>` invokes the entrypoint directly. SANCHO-scheduled behavior requires a `sancho-task` kind plugin.
</Tip>

## Portable vs internal skills

Some skills ship a portable `SKILL.md` that external AI runtimes (LangChain, OpenAI Assistants, Cursor rules) can read and wire in without a running Makakoo install. Others are internal-only — they depend on the Brain, Olibia, SANCHO, or HarveyChat and have no meaning outside a live Makakoo install.

Use `makakoo skill skill_discover` (from `lib-harvey-core`) to walk the installed tree and discover every `SKILL.md`:

```bash
makakoo skill skill_discover
```

## Discover skills at runtime

The `skill_discover` MCP tool (from `lib-harvey-core`) walks `$MAKAKOO_HOME/plugins/*/SKILL.md` and returns every portable skill file. Use it inside a Harvey session to find skills relevant to your current task:

```bash
# Install the core library (ships skill_discover)
makakoo plugin install lib-harvey-core --core

# Discover all installed SKILL.md files
makakoo skill skill_discover
```
