The `makakoo` binary is the single entry point for the entire Makakoo OS. Running `makakoo` with no arguments prints a first-run banner. Every feature ŌĆö Brain search, CLI infection, daemon control, plugin management, write-access grants ŌĆö is exposed as a subcommand. This page lists every top-level subcommand with a one-line description and links to the detailed reference pages where they exist.

## Global behavior

Running `makakoo` with no subcommand prints a first-run banner instead of an error. Every subcommand supports `--help` for a usage summary.

```bash
makakoo --help
makakoo search --help
makakoo plugin install --help
```

## Exit codes

| Code | Meaning |
|---|---|
| `0` | Success |
| `1` | General error or drift detected (e.g. `infect --verify` exits 1 when any CLI has drifted) |
| `2` | JSON response validation failed (e.g. `run --json` when response is not JSON) |
